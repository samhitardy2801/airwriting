# AirWriting ✨

An AI-Based Virtual Writing System Using Hand Gestures

## 📌 Overview

AirWriting is a computer vision-based project that enables users to write or draw in the air using only hand gestures captured through a webcam. The system tracks finger movements in real time and displays the written content digitally on the screen without requiring a keyboard, mouse, or touch device.

This project creates a touchless and interactive writing experience by combining Artificial Intelligence, Hand Tracking, and Gesture Recognition technologies. It demonstrates how human-computer interaction can be improved using AI-powered vision systems.

---

## 🚀 Features

* ✋ Real-time hand and finger tracking
* 🖊️ Write or draw in the air using gestures
* 📷 Webcam-based interaction
* ⚡ Smooth and responsive virtual writing
* 🧠 AI-powered gesture recognition
* 💻 User-friendly interface
* ❌ No physical contact required

---

## 🛠️ Technologies Used

* **Python** – Core programming language
* **OpenCV** – Image processing and webcam handling
* **MediaPipe** – Hand tracking and landmark detection
* **NumPy** – Numerical operations and array processing

---

## 📂 Project Workflow

1. Capture live video from webcam
2. Detect hand landmarks using MediaPipe
3. Track finger movement positions
4. Draw movement trajectory on virtual canvas
5. Display real-time digital writing output
